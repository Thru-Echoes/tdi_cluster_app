Predict a Park's Crime Profile


Enter metrics for a new park or an existing park that is changing. After entering values, click the predict crime profile button to see the likelihood that this new or changed park will belong to each of the seven Crime Profile clusters. The crime profile clusters of the 30 PWR park units are shown below for comparison.

Explore Park Metrics:

The total number of rangers is calculated by adding the number of rangers and the number of seasonal rangers at peak season. Thus, total number of rangers = number of rangers + number of seasonal rangers. For example, if a park has 5 rangers and 2 seasonal rangers, 7 total number of rangers would be shown in the table below.


The number of rangers and the number of seasonal rangers, as well as all other park metrics used in this web tool, are each a single value that represents the average of that parameter in a park across 10 years of data collection.
